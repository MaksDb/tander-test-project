This project was created in IDE Intellij IDEA, and you can launch it in the same IDE.
In catalog out/artifacts/tander_test_project_jar you can find executable jar file for quickly launching from command line.
Use the command "java -jar tander-test-project.jar"
After the startup, enter the integer N.
Result of the program's work is 2 xml files(1.xml and 2.xml) and summ of all numbers from "2.xml"
